# Tesla (TSLA) Stock Performance & Volatility Analysis

Project Overview
This project is an interactive financial data analysis tool focused on Tesla's (TSLA) stock performance from 2021 to 2026. The goal is to bridge the gap between raw market data and actionable insights by using Python to visualize trends and calculate key risk metrics. This was developed as a "production-ready" interactive notebook for financial data exploration.

Key Features
Automated Data Retrieval: Fetches real-time and historical stock data using the 'efinance' library.
Technical Indicators: Implementation of 20-day and 50-day Moving Averages (MA) to filter market noise and identify long-term momentum.
Volatility Visualization: Custom plotting of daily returns to visualize market swings and high-risk periods.
Descriptive Statistics: Comprehensive statistical summary including mean, standard deviation, and extreme value identification (finding the exact dates of max gains/losses).
Interactive Search: A user-friendly function allowing investors to query specific returns for any given historical date.

Tech Stack
Language: Python
Libraries: Pandas (Data cleaning), Seaborn (Visualization), Efinance (Data source).
Platform: GitHub Interactive Notebook (.ipynb)
